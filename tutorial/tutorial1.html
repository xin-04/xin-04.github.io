Fundamentals

  1. Explain the concept of responsive web design.

    • To create a website that looks good on all devices which changes its layout based on the size and capabilities of the device.

  2. State three important elements of a website design.

    • Navigation: website should be easy to navigate and menu should be easily accessible.
    • Visual Design: website should be organized in a way that makes sense to the viewers.
    • Speed: users should receive real-time information from the website.

  3. List the basic step of website design process.

    1. Planning: set goals, research and gather contents
    2. Designing: set a prototype, outline the web page elements
    3. Coding: HTML/CSS, JavaScript
    4. Testing: improve website, maintain the website to be error-free and control the quality of the website
    5. Launch: publish website to WWW
    6. Maintenance: future improvement and enhancement of the website

  4. State TWO (2) differences between HTTP and HTTPS.
    5. HTTP HTTPS
    Hypertext Transfer Protocol Hypertext Transfer Protocol Secure
    written as http:// written as https://
    used at the application layer used at the transport layer
    insecure because it sends plain data secure because it sends encrypted data

  5. State TWO (2) differences between static website and dynamic website.
    6. Static Website Dynamic Website
    Is prebuilt source code Is not prebuilt source code
    Utilize HTML, CSS, JavaScript (does not interact with database) Utilize HTML, CSS, JavaScript, PHP, SQL (interact with database)
    Does not process content on the server Process content on the server
    Executed by user's browser Executed by server
    Displays the same information to all users Displays different information to different users
    Is not interactable for users Interactable for users
    Faster (returned by server without change) Slower (has update and changes)
    Only website developer makes changes in the website Both user and website developer can make changes in the website
